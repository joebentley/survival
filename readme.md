# Survival (WIP)

A roguelike survival game inspired by Dwarf Fortress, Caves of Qud, and Don't Starve

To build:

```
$ mkdir build
$ cd build
$ cmake ..
$ make
```

Build requirements: `cmake`, `Boost`, `SDL2`, and `SDL2_image`
